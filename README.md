# 🌐 Branding Website – Contact Page Project

A modern, responsive branding/agency website built using **HTML5, CSS3, and JavaScript**, featuring a structured Contact page with a clean form layout and professional UI design.

This project focuses on building a multi-page website with organized structure, responsive layout principles, and user-friendly form design.

---

## 📌 Project Overview

This is a multi-page branding website that includes:

- 🏠 Home Page  
- ℹ️ About Page  
- 📬 Contact Page  

The Contact page allows users to:

- Enter First Name  
- Enter Last Name  
- Provide Email Address  
- Write a Message  
- Submit the form using a “Send Message” button  

The page also displays structured contact information including:

- 📞 Phone number  
- 📧 Email address  
- 📍 Physical location  

This project was built to strengthen understanding of layout structuring, form design, and responsive UI development.
